## 10.1 Computer Virus ဆိုတာ ဘာလဲ

**Computer Virus** ဆိုတာ
👉 **ကွန်ပျူတာထဲကို ဝင်ပြီး system ကို ထိခိုက်စေတဲ့ အန္တရာယ်ရှိတဲ့ program** ဖြစ်ပါတယ်။

### Virus ဝင်လာတဲ့ လမ်းကြောင်းများ

* Unknown USB
* Suspicious email attachment
* Pirated software
* Fake website links

📌 **Teacher Note**

> Virus = လူ့ကိုယ်ထဲဝင်တဲ့ ရောဂါလိုပဲ

---

## 10.2 Malware အမျိုးအစားများ (အလွယ်ဆုံး)

* **Virus** – File ကိုဖျက်
* **Worm** – ကိုယ်တိုင်ပြန့်ပွား
* **Trojan** – ကောင်းတာလိုပြပြီး မကောင်းလုပ်
* **Spyware** – ကိုယ်ရေးကိုယ်တာ data ခိုး

📌 Beginner အတွက် အမည်လောက်မှတ်ရင်လုံလောက် 👍

---

## 10.3 Antivirus Software

**Antivirus** ဆိုတာ
👉 **Virus တွေကို စစ်ဆေး၊ ကာကွယ်ပေးတဲ့ software**

### Examples

* Windows Defender
* Avast
* Kaspersky

### Basic Rules

* Antivirus install လုပ်ထား
* Update အမြဲလုပ်
* Scan ပုံမှန်လုပ်

---

## 10.4 Password Security (အရမ်းအရေးကြီး ⚠️)

### Strong Password ဘယ်လိုလုပ်မလဲ

* အနည်းဆုံး 8 လုံး
* Capital + Small letters
* Number + Symbol ပါ

❌ 123456
❌ password
✅ Z@ne2026!

📌 **Golden Rule**

> Password ကို မည်သူ့ကိုမှ မပေးပါ ❗

---

## 10.5 Phishing (လူလိမ် Online)

**Phishing** ဆိုတာ
👉 **Fake email / website နဲ့ password, money ခိုးတဲ့ လုပ်ရပ်**

### Phishing ကို ခွဲခြားနည်း

* “Urgent” လို့ပြော
* Link ကိုနှိပ်ခိုင်း
* Free prize ဆိုပြီး ဆွဲဆောင်

📌 **Teacher Warning**

> မသေချာရင် **မနှိပ်ပါ** ❌

---

## 10.6 Internet Safety Rules

### Do ✅

* Trusted websites သာဝင်
* Logout ပြီးမှ ထွက်
* Public computer မှာ data မသိမ်း

### Don’t ❌

* Personal info မမျှဝေ
* Unknown link မနှိပ်
* Pirated software မသုံး

---

## 10.7 Computer Ethics (ယဉ်ကျေးမှု)

**Computer Ethics** ဆိုတာ
👉 **ကွန်ပျူတာကို မှန်ကန်စွာ၊ တာဝန်ယူပြီး သုံးခြင်း**

### Ethics Examples

* Software piracy မလုပ်
* တခြားသူ data မခိုး
* Online bullying မလုပ်
* Copyright ကိုလေးစား

📌 **Teacher Quote**

> “Good computer user = Good digital citizen”

---

## 10.8 Common Mistakes (အများဆုံးတွေ့ရ)

* Antivirus မတင်
* Password တစ်ခုကို နေရာတိုင်းသုံး
* Free software အမည်ကြီးဆိုပြီး ယုံ
* Internet = safe လို့ ထင်
