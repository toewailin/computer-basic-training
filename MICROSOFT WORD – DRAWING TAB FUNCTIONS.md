# ✏️ MS WORD 2021 – DRAWING TAB FUNCTIONS

(English → မြန်မာလို အဓိပ္ပါယ်)

---

## 🔹 1. Drawing Tools (Pen Tools)

| English Function | မြန်မာလို Meaning          |
| ---------------- | -------------------------- |
| Draw             | လက်ရေးဆွဲခြင်း             |
| Pen              | ဘောပင်နဲ့ရေးဆွဲခြင်း       |
| Pencil           | ခဲတံလိုရေးဆွဲခြင်း         |
| Highlighter      | စာတောက်အောင် မျဉ်းဆွဲခြင်း |
| Add Pen          | ဘောပင်အသစ်ထည့်ခြင်း        |

📌 **Teacher Note**

> Touch screen / mouse နဲ့ လက်ရေးဆွဲနိုင်ပါတယ်

---

## 🔹 2. Pen Options

| Function      | မြန်မာလို Meaning  |
| ------------- | ------------------ |
| Pen Color     | ဘောပင်အရောင်ရွေး   |
| Pen Thickness | ဘောပင်အထူပြောင်း   |
| Pen Style     | ဘောပင်ပုံစံပြောင်း |

---

## 🔹 3. Erase Tools

| English Function | မြန်မာလို Meaning      |
| ---------------- | ---------------------- |
| Eraser           | ဖျက်တံ                 |
| Stroke Eraser    | ဆွဲထားတဲ့မျဉ်းကို ဖျက် |
| Point Eraser     | အစိတ်အပိုင်းဖျက်       |

📌 **Rule**

> Eraser = လက်ရေး/ပုံဆွဲထားတာ ဖျက်ဖို့ပဲ သုံး

---

## 🔹 4. Convert Tools

| English Function | မြန်မာလို Meaning                |
| ---------------- | -------------------------------- |
| Ink to Shape     | လက်ရေးကို ပုံသဏ္ဍာန်ပြောင်း      |
| Ink to Text      | လက်ရေးစာကို စာသားပြောင်း         |
| Ink to Math      | လက်ရေးသင်္ချာကို formula ပြောင်း |

📌 **Example**

> လက်ရေးနဲ့ circle ဆွဲ → shape အဖြစ်ပြောင်း

---

## 🔹 5. Lasso Select

| English Function | မြန်မာလို Meaning             |
| ---------------- | ----------------------------- |
| Lasso Select     | လက်ရေးကို ဝိုင်းရွေးချယ်ခြင်း |

📌 **Use**

> ဆွဲထားတဲ့စာ/ပုံတွေကို ရွေးပြီး ပြင်ဖို့

---

## 🔹 6. Ruler Tool

| English Function | မြန်မာလို Meaning      |
| ---------------- | ---------------------- |
| Ruler            | အတိုင်းအတာနဲ့ မျဉ်းဆွဲ |
| Rotate Ruler     | တန်းကို လှည့်ခြင်း     |

📌 **Teacher Tip**

> တိတိကျကျ မျဉ်းဆွဲချင်ရင် Ruler သုံး

## 📝 Short Exam Notes (မှတ်ထားရန်)

* **Pen / Pencil** → လက်ရေးဆွဲ
* **Eraser** → လက်ရေးဖျက်
* **Ink to Text** → လက်ရေး → စာသား
* **Ink to Shape** → လက်ရေး → ပုံသဏ္ဍာန်
* **Ruler** → မျဉ်းတိတိကျကျဆွဲ
