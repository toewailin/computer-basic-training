# 📊 EXCEL FORMULAS & FUNCTIONS (MYANMAR MEANING)

---

## 🔹 1. Basic Math Functions (အခြေခံတွက်ချက်)

| Formula       | မြန်မာလို Meaning       |
| ------------- | ----------------------- |
| `SUM()`       | စုစုပေါင်းတွက်          |
| `AVERAGE()`   | ပျမ်းမျှတွက်            |
| `MAX()`       | အမြင့်ဆုံးတန်ဖိုး       |
| `MIN()`       | အနိမ့်ဆုံးတန်ဖိုး       |
| `PRODUCT()`   | မြှောက်စု               |
| `ROUND()`     | ကိန်းဂဏန်းဝိုင်း        |
| `ROUNDUP()`   | အပေါ်ဘက်ဝိုင်း          |
| `ROUNDDOWN()` | အောက်ဘက်ဝိုင်း          |
| `ABS()`       | အနုတ်ကို အပေါင်းပြောင်း |
| `INT()`       | ကိန်းပြည့်သာယူ          |
| `MOD()`       | စားလဒ်အကြွင်း           |

---

## 🔹 2. Counting Functions (ရေတွက်မှု)

| Formula        | မြန်မာလို Meaning        |
| -------------- | ------------------------ |
| `COUNT()`      | ကိန်းဂဏန်းပါ Cell ရေတွက် |
| `COUNTA()`     | အချက်အလက်ပါ Cell ရေတွက်  |
| `COUNTBLANK()` | အလွတ် Cell ရေတွက်        |
| `COUNTIF()`    | အခြေအနေတစ်ခုနဲ့ ရေတွက်   |
| `COUNTIFS()`   | အခြေအနေများနဲ့ ရေတွက်    |

---

## 🔹 3. Logical Functions (အခြေအနေအလိုက်)

| Formula     | မြန်မာလို Meaning        |
| ----------- | ------------------------ |
| `IF()`      | အခြေအနေစစ်ပြီး အဖြေပေး   |
| `IFS()`     | အခြေအနေများစစ်           |
| `AND()`     | အားလုံးမှန်ရင်မှန်       |
| `OR()`      | တစ်ခုခုမှန်ရင်မှန်       |
| `NOT()`     | မှန်/မှား ပြောင်း        |
| `IFERROR()` | Error ဖြစ်ရင် အစားထိုးပြ |

---

## 🔹 4. Lookup & Reference Functions (ရှာဖွေမှု)

| Formula     | မြန်မာလို Meaning    |
| ----------- | -------------------- |
| `VLOOKUP()` | ဒေါင်လိုက်ရှာ        |
| `HLOOKUP()` | အလျားလိုက်ရှာ        |
| `XLOOKUP()` | အဆင့်မြင့်ရှာဖွေ     |
| `LOOKUP()`  | တန်ဖိုးရှာ           |
| `INDEX()`   | နေရာအလိုက် တန်ဖိုးယူ |
| `MATCH()`   | တည်နေရာရှာ           |
| `OFFSET()`  | ရွှေ့ပြီးယူ          |

---

## 🔹 5. Text Functions (စာသားဆိုင်ရာ)

| Formula        | မြန်မာလို Meaning   |
| -------------- | ------------------- |
| `LEFT()`       | ဘယ်ဘက်စာယူ          |
| `RIGHT()`      | ညာဘက်စာယူ           |
| `MID()`        | အလယ်စာယူ            |
| `LEN()`        | စာလုံးရေအရေအတွက်    |
| `CONCAT()`     | စာသားပေါင်း         |
| `TEXTJOIN()`   | အခြေအနေပါ စာပေါင်း  |
| `UPPER()`      | စာအကြီးပြောင်း      |
| `LOWER()`      | စာအသေးပြောင်း       |
| `PROPER()`     | စာလုံးအစအကြီး       |
| `TRIM()`       | အပို Space ဖျက်     |
| `REPLACE()`    | နေရာအလိုက် အစားထိုး |
| `SUBSTITUTE()` | စာလုံးအစားထိုး      |
| `TEXT()`       | Format ပြောင်း      |
