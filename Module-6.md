## 6.1 Internet ဆိုတာ ဘာလဲ

**Internet** ဆိုတာ
👉 **ကမ္ဘာအနှံ့က ကွန်ပျူတာများကို ချိတ်ဆက်ထားတဲ့ network အကြီးကြီးတစ်ခု** ဖြစ်ပါတယ်။

### Internet ကို ဘာအတွက်သုံးလဲ

* အချက်အလက်ရှာဖွေ (Search)
* Email ပို့/လက်ခံ
* Social media သုံး
* Online learning
* Online banking / shopping

📌 **Teacher Note (အလွယ်မှတ်ရန်)**

> Internet = ကမ္ဘာလုံးဆိုင်ရာ network
> Wi-Fi / Mobile data = Internet ကို ချိတ်ဖို့ အသုံးပြုတဲ့ လမ်းကြောင်း

---

## 6.2 Internet vs WWW (World Wide Web)

ဒီအပိုင်းမှာ ကျောင်းသားအများဆုံး မှားတတ်တယ် ⚠️

| Internet      | World Wide Web (WWW)       |
| ------------- | -------------------------- |
| Network       | Websites                   |
| အခြေခံ        | Internet ပေါ်မှာ အလုပ်လုပ် |
| Email, FTP ပါ | Websites သာ                |

📌 **Example**

* Internet = လမ်းကြီး
* Website = လမ်းပေါ်က အိမ်တွေ

---

## 6.3 Web Browser ဆိုတာ ဘာလဲ

**Web Browser** ဆိုတာ
👉 **Website တွေကို ဖွင့်ကြည့်ဖို့ အသုံးပြုတဲ့ software** ဖြစ်ပါတယ်။

### အသုံးများတဲ့ Browsers

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

📌 **Teacher Demo**

1. Browser ဖွင့်ပါ
2. Address bar ကို ပြပါ
3. Cursor blinking ဖြစ်နေတာကို သေချာပြ

---

## 6.4 Website, Webpage & URL

### Website

* Page အများကြီးပါသော site တစ်ခု
* Example: facebook.com

### Webpage

* Website ထဲက စာမျက်နှာတစ်ခု
* Example: facebook.com/login

### URL (Website Address)

* Website ရဲ့ လိပ်စာ
* Example: [https://www.google.com](https://www.google.com)

📌 **Teacher Trick**

> URL ကို “အိမ်လိပ်စာ” လို့ ခိုင်းနှိုင်းရှင်းပြ

---

## 6.5 Search Engine ဆိုတာ ဘာလဲ

**Search Engine** ဆိုတာ
👉 **Internet ထဲက အချက်အလက်တွေကို ရှာဖွေပေးတဲ့ tool** ဖြစ်ပါတယ်။

### အသုံးများတဲ့ Search Engines

* Google ✅
* Bing
* Yahoo

---

## 6.6 Google Search အသုံးပြုနည်း (အရေးကြီးဆုံး)

### Step-by-step (လက်တွေ့)

1. Browser ဖွင့်
2. [www.google.com](http://www.google.com) ဝင်
3. Search box ထဲမှာ စာရိုက်
4. Enter နှိပ်

### Good Search Keywords (Professional Tip)

❌ “computer”
✅ “computer basic course for beginners pdf”

📌 **Rule**

> Keywords များလေလေ → Result ပိုတိကျလေလေ

## 6.7 Internet Services (အခြေခံ)

* Email
* Social Media
* Cloud Storage (Google Drive)

---

## 6.8 Internet Safety (Beginner Level)

### Do ✅

* Strong password သုံး
* Trusted websites သာ ဝင်
* Logout လုပ်ပြီးထွက်

### Don’t ❌

* Unknown links မနှိပ်
* Personal info မမျှဝေ
* Password မပေး

📌 **Golden Rule**

> “Internet ပေါ်တင်လိုက်တဲ့အရာ မပျောက်နိုင်”

---

## 6.9 Common Mistakes (ဆရာတစ်ယောက်အနေနဲ့ သတိပေး)

* Address bar နဲ့ Search box မခွဲနိုင်ခြင်း
* Fake website ကို ယုံခြင်း
* Caps Lock မသိဘဲ password မှားခြင်း
* Internet မရှိဘဲ browser ပျက်တယ်ထင်ခြင်း
