## 🔹 1. File Menu Functions

## 🎓 Teacher Exam Summary (အရမ်းအရေးကြီး ⚠️)

* **New Slide / Layout** → Slide ဖန်တီး
* **Design / Theme** → Presentation အလှ
* **Transition vs Animation** → မေးခွန်းအမြဲပါ
* **Slide Show** → Presentation ပြသ

---

## 🧠 မှတ်သားရန် (၁ကြိမ်ကြည့်ပြီး မှတ်မိအောင်)

* Transition = Slide ပြောင်း effect
* Animation = Object လှုပ် effect
* Theme = Slide အလှပုံစံ
* Slide Show = ပြသခြင်း
