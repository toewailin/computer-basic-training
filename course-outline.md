# 🖥️ COMPUTER BASIC COURSE OUTLINE

## 📘 Course Objective

ဒီသင်တန်းပြီးရင် ကျောင်းသားက

* ကွန်ပျူတာကို မကြောက်တော့ဘူး
* ကိုယ်တိုင်ဖွင့်၊ ပိတ်၊ သုံး၊ ဖိုင်တွေစီမံနိုင်မယ်
* Internet, Email, Word/Excel ကို အခြေခံအဆင့်သုံးနိုင်မယ်

---

## 🔹 Module 1: Introduction to Computer

**(ကွန်ပျူတာအခြေခံအကြောင်း သိစေခြင်း)**

* Computer ဆိုတာဘာလဲ
* Computer ကို ဘာအတွက်သုံးလဲ
* Computer ရဲ့ အမျိုးအစားများ

  * Desktop
  * Laptop
  * Tablet
  * Smartphone
* Hardware vs Software (အလွယ်ဆုံးရှင်းပြ)

📌 *Outcome:*
👉 Computer ဆိုတာဘာလဲ ကိုယ်ပိုင်စကားနဲ့ ပြောနိုင်မယ်

---

## 🔹 Module 2: Computer Hardware

**(ကွန်ပျူတာအစိတ်အပိုင်းများ)**

* Input Devices

  * Keyboard
  * Mouse
  * Scanner
* Output Devices

  * Monitor
  * Printer
  * Speaker
* System Unit အတွင်းပိုင်း

  * CPU (Brain of Computer)
  * RAM (Memory)
  * Hard Disk / SSD (Storage)

📌 *Outcome:*
👉 Hardware တစ်ခုချင်းစီရဲ့ အလုပ်ကို ပြောနိုင်မယ်

---

## 🔹 Module 3: Software & Operating System

**(Software နဲ့ OS ကို နားလည်စေခြင်း)**

* Software ဆိုတာဘာလဲ
* System Software vs Application Software
* Operating System ဆိုတာဘာလဲ

  * Windows
  * macOS
  * Linux
* OS ရဲ့ အလုပ်လုပ်ပုံ အခြေခံ

📌 *Outcome:*
👉 OS မရှိရင် computer မအလုပ်လုပ်ဘူးဆိုတာ နားလည်မယ်

---

## 🔹 Module 4: Basic Computer Operations

**(ကွန်ပျူတာအသုံးပြုနည်း အခြေခံ)**

* Computer ဖွင့်နည်း / ပိတ်နည်း
* Mouse အသုံးပြုနည်း

  * Click
  * Double click
  * Right click
  * Drag & Drop
* Keyboard အသုံးပြုနည်း

  * Alphabet keys
  * Number keys
  * Special keys (Enter, Backspace, Shift)

📌 *Outcome:*
👉 Mouse / Keyboard ကို သေချာသုံးနိုင်မယ်

---

## 🔹 Module 5: File & Folder Management

**(ဖိုင်တွေကို စနစ်တကျ စီမံနည်း)**

* File ဆိုတာဘာလဲ
* Folder ဆိုတာဘာလဲ
* File Explorer အသုံးပြုနည်း
* Folder အသစ်ဖန်တီးနည်း
* File ကို

  * Save
  * Rename
  * Copy / Paste
  * Delete

📌 *Outcome:*
👉 ဖိုင်ပျောက်တာ မဖြစ်တော့ဘူး 😄

---

## 🔹 Module 6: Introduction to Internet

**(အင်တာနက် အခြေခံနားလည်မှု)**

* Internet ဆိုတာဘာလဲ
* Web Browser ဆိုတာဘာလဲ

  * Chrome
  * Edge
  * Firefox
* Website, URL ဆိုတာဘာလဲ
* Google Search သုံးနည်း

📌 *Outcome:*
👉 ကိုယ်လိုချင်တဲ့အချက်အလက်ကို Google နဲ့ရှာနိုင်မယ်

---

## 🔹 Module 7: Email Basics

**(Email အသုံးပြုနည်း)**

* Email ဆိုတာဘာလဲ
* Gmail account ဖန်တီးနည်း
* Email ပို့နည်း
* Email ဖတ်နည်း
* Attachment ပို့/လက်ခံနည်း

📌 *Outcome:*
👉 Professional Email အသုံးပြုနိုင်မယ်

---

## 🔹 Module 8: Microsoft Word (Basic)

**(စာရိုက်အခြေခံ)**

* Microsoft Word ဖွင့်နည်း
* Text ရိုက်နည်း
* Font size, Font style ပြောင်းနည်း
* Paragraph alignment
* Save / Print

📌 *Outcome:*
👉 Letter, Assignment ရိုက်နိုင်မယ်

---

## 🔹 Module 9: Microsoft Excel (Basic)

**(ဇယားအခြေခံ)**

* Workbook / Worksheet
* Cell, Row, Column
* Data ထည့်နည်း
* Basic formulas

  * SUM
  * AVERAGE
* Simple table ဖန်တီးခြင်း

📌 *Outcome:*
👉 အမှတ်စာရင်း၊ စာရင်းဇယား လုပ်နိုင်မယ်

---

## 🔹 Module 10: Computer Safety & Ethics

**(လုံခြုံရေးနဲ့ ယဉ်ကျေးမှု)**

* Virus ဆိုတာဘာလဲ
* Antivirus အခြေခံ
* Password လုံခြုံရေး
* Internet Ethics
* Cyber Safety အခြေခံ

📌 *Outcome:*
👉 ကိုယ့် data ကို ကိုယ်ကာကွယ်နိုင်မယ်

---

## 🎓 Course Completion Outcome

သင်တန်းပြီးရင် ကျောင်းသားက

* Computer ကို confidence နဲ့ သုံးနိုင်မယ်
* Office / School / Daily life မှာ အသုံးချနိုင်မယ်
* Advanced Computer / IT course တွေ ဆက်လေ့လာဖို့ အဆင်သင့်ဖြစ်မယ်
