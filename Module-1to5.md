### 1.1 Computer ဆိုတာ ဘာလဲ

**Computer** ဆိုတာ
👉 **အချက်အလက်တွေကို လက်ခံ → စဉ်းစား → အဖြေထုတ်ပေးနိုင်တဲ့ electronic machine** ဖြစ်ပါတယ်။

📌 **အလွယ်မှတ်ရန်**

> Computer = Input → Process → Output

### 1.2 Computer ကို ဘာအတွက် သုံးလဲ

* စာရေး၊ စာရိုက်
* အချက်အလက်ရှာ
* Email / Internet သုံး
* Video ကြည့်
* Online သင်ကြားရေး

### 1.3 Computer အမျိုးအစားများ

* Desktop
* Laptop
* Tablet
* Smartphone

📌 **Teacher Note**

> Phone လည်း computer အမျိုးအစားတစ်ခုပါ 👍

### 1.4 Hardware vs Software

* **Hardware** → လက်နဲ့ထိလို့ရ (Keyboard, Mouse)
* **Software** → မထိလို့ရ (Windows, Word)

### 2.1 Hardware ဆိုတာ ဘာလဲ

👉 **Computer ကို ဖွဲ့စည်းထားတဲ့ အစိတ်အပိုင်းတွေ**

---

### 2.2 Input Devices (အချက်အလက်ထည့်)

* Keyboard
* Mouse
* Scanner
* Microphone

📌 **Input = ထည့်**

---

### 2.3 Output Devices (အဖြေထုတ်)

* Monitor
* Printer
* Speaker

📌 **Output = ထွက်**

---

### 2.4 System Unit (အရေးကြီးဆုံး)

* **CPU** → ဦးနှောက်
* **RAM** → မှတ်ဉာဏ် (အလုပ်လုပ်နေချိန်)
* **Hard Disk / SSD** → သိမ်းဆည်းရာ

📌 **Teacher Trick**

> RAM မရှိရင် computer နှေး
> Disk မရှိရင် data မသိမ်းနိုင်
