## 9.1 Microsoft Excel ဆိုတာ ဘာလဲ

**Microsoft Excel** ဆိုတာ
👉 **ဇယားများ၊ အမှတ်စာရင်းများ၊ စာရင်းတွက်ချက်မှုများအတွက် အသုံးပြုတဲ့ software** ဖြစ်ပါတယ်။

### Excel ကို ဘာအတွက်သုံးလဲ

* ကျောင်းသားအမှတ်စာရင်း
* ငွေစာရင်း
* Attendance
* Data စုစည်းခြင်း

📌 **Teacher Note**

> Word = စာရေး
> Excel = စာရင်းတွက်

---

## 9.2 Workbook & Worksheet

* **Workbook** → Excel file တစ်ခုလုံး
* **Worksheet** → Workbook ထဲက စာရွက်တစ်ရွက်

📌 Excel ဖွင့်လိုက်ရင် **Sheet1** ကို တွေ့ရမယ်

---

## 9.3 Row, Column & Cell

* **Column** → အပေါ်မှအောက် (A, B, C…)
* **Row** → ဘယ်မှညာ (1, 2, 3…)
* **Cell** → Row + Column ဆုံရာ

  * Example: **A1, B2**

📌 **Golden Rule**

> Excel မှာ အလုပ်အားလုံးက Cell ကနေစ

---

## 9.4 Data ထည့်နည်း

1. Cell တစ်ခု click
2. Keyboard နဲ့စာရိုက်
3. Enter နှိပ်

### Example Table

| A    | B    | C       |
| ---- | ---- | ------- |
| Name | Math | English |
| Aung | 75   | 80      |

## 9.6 Formula ဆိုတာ ဘာလဲ

**Formula** ဆိုတာ
👉 **Excel ကို တွက်ချက်ခိုင်းတဲ့ command** ဖြစ်ပါတယ်။

📌 Formula အားလုံးက **=** နဲ့ စရပါတယ် ❗

---

## 9.7 SUM Function (အများဆုံးသုံး)

**SUM** = စုစုပေါင်းတွက်

### Example

```
=SUM(B2:B4)
```

👉 B2 မှ B4 အထိ စုစုပေါင်း

📌 **Teacher Trick**

> Mouse နဲ့ drag လုပ်ပြီး range ရွေးရင် အမှားနည်း

---

## 9.8 AVERAGE Function

**AVERAGE** = ပျမ်းမျှတွက်

### Example

```
=AVERAGE(C2:C4)
```

📌 Exam မှာ မကြာခဏမေး ⚠️

---

## 9.9 AutoFill (အရမ်းအသုံးဝင်)
