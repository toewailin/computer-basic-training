## 8.1 Microsoft Word ဆိုတာ ဘာလဲ

**Microsoft Word** ဆိုတာ
👉 **စာရိုက်၊ စာပြင်၊ စာရွက်စာတမ်းဖန်တီးဖို့ အသုံးပြုတဲ့ software** ဖြစ်ပါတယ်။

### Word ကို ဘာအတွက်သုံးလဲ

* စာရေးစာချွန် (Letter)
* ကျောင်းစာအလုပ် (Assignment)
* ကိုယ်ရေးရာဇဝင် (CV)
* အစီရင်ခံစာ (Report)

📌 **ဆရာမှတ်ချက်**

> Word = Digital စာရေးစာရွက်

---

## 8.2 Microsoft Word ဖွင့်နည်း

### နည်းလမ်း (၂) မျိုး

1. Start Menu → Word
2. Desktop ရှိ Word icon ကို double click

ဖွင့်လိုက်ရင် **Blank document** ကိုရွေးပါ။

---

## 8.3 Word Screen ကို နားလည်ခြင်း

အရေးကြီးတဲ့နေရာများ—

* **Title Bar** – ဖိုင်နာမည်
* **Ribbon** – Tools တွေရှိတဲ့နေရာ
* **Document Area** – စာရိုက်တဲ့နေရာ
* **Cursor** – စာရိုက်မည့်နေရာကိုပြတဲ့မျဉ်း

📌 Cursor မရှိရင် စာမရိုက်နိုင်ပါ ❗

---

## 8.4 စာရိုက်နည်း (Typing)

* Keyboard နဲ့ စာရိုက်
* Enter → လိုင်းအသစ်
* Backspace → နောက်စာဖျက်
* Space → စာလုံးကြားခြား

📌 **လက်တွေ့**

> ကိုယ့်နာမည် + ဒီနေ့ရက်စွဲ ကို ရိုက်ကြည့်ပါ

---

## 8.5 Font & Font Size ပြင်နည်း

### Font

* စာလုံးပုံစံ (ဥပမာ Times New Roman)

### Font Size

* စာလုံးအရွယ် (ဥပမာ 12, 14)

### ပြင်နည်း

1. စာကို select လုပ်
2. Home tab → Font / Size ပြောင်း

📌 **အလုပ်စာအတွက်**

* Font Size 12
* Clear font သုံး

---

## 8.6 Text Formatting (စာလုံးပြင်ဆင်ခြင်း)

* **Bold** – စာအထူ
* **Italic** – စာစောင်း
* **Underline** – စာအောက်မျဉ်း

📌 **မလိုအပ်ရင် မသုံးပါ** (စာမသန့်ဘူး)

---

## 8.7 Paragraph Alignment

* **Left** – ဘယ်ဘက်ညှိ
* **Center** – အလယ်ညှိ
* **Right** – ညာဘက်ညှိ
* **Justify** – နှစ်ဖက်ညှိ (စာတမ်းအတွက်ကောင်း)

📌 **Assignment / Report**

> Justify သုံးရင် အလှဆုံး 👍

---

## 8.8 File Save နည်း

### Save Steps

1. File → Save As
2. Folder ရွေး
3. File name ရိုက်
4. Save

📌 **ဆရာသတိပေး**

> Save မလုပ်ရင် မီးပျက်တာနဲ့ စာပျောက် 😄

---

## 8.9 Print နည်း

1. File → Print
2. Printer ရွေး
3. Pages စစ်
4. Print

📌 Print မလုပ်ခင် **Preview** ကိုကြည့်ပါ

---

## 8.10 Common Mistakes (အများဆုံးမှားတတ်)

* Save မလုပ်ဘဲပိတ်
* Font မညီမညာ
* Space ကို Enter နဲ့သုံး
* Heading ကို Bold မပြုလုပ်
